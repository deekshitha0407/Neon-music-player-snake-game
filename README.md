# Neon Music Player & Snake Game

A retro-futuristic web application that combines a Neon-themed Music Player with a classic Snake Game. This project demonstrates creative UI design along with interactive gameplay using modern web technologies.

## Project Overview
The application features a cyberpunk-inspired interface with glowing neon elements. Users can listen to music while playing a classic snake game within the same interface.

## Features
- Neon cyberpunk UI design
- Integrated music player
- Classic snake game
- Real-time score tracking
- Restart game functionality
- Keyboard controls using Arrow keys or WASD
- Animated glowing effects and modern UI styling

## Technologies Used
- HTML5
- CSS3
- JavaScript
- Canvas API

## How to Run the Project
1. Download or clone this repository.
2. Navigate to the project folder.
3. Open the **index.html** file in your browser.
4. The application will start automatically.

## Game Controls
- **Arrow Keys / WASD** → Move the snake
- **Play Button** → Start music
- **Pause Button** → Pause music
- **Reboot System** → Restart the game
